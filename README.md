# Stock Profit Analyzer

This is a beginner Python finance project that analyzes multiple stocks and calculates:

- Profit/Loss
- Percentage Returns
- Trade Performance

## Features
- Uses Python functions
- Uses loops and dictionaries
- Handles profitable and loss-making trades
- Calculates percentage returns

## Technologies Used
- Python
